# JobTracker — Suivi des candidatures

Application pour gérer et suivre vos candidatures.

## Structure du projet

```
jobtracker/
├── client/                 # Application React (Vite)
├── server/                 # API Express
└── docs/                   # Documentation
```

## Installation

```bash
pnpm install
```

## Démarrer

```bash
pnpm dev
```
